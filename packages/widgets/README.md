# @pilates/widgets

Interactive widgets for [Pilates](https://github.com/pilatesjs/pilates) terminal UIs: `<TextInput>`, `<Select>`, `<Spinner>`.

Built on [`@pilates/react`](https://www.npmjs.com/package/@pilates/react). Zero runtime dependencies.

```sh
npm install @pilates/widgets @pilates/react react
```

Usage examples coming with the v0.1.0 release.
